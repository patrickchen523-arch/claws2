# 长期记忆

## 游戏拆解分析方法论

### X-Statement 公式模板
> [题材][核心玩法]融合[次级玩法]与[养成要素]，玩家操控[角色类型]，在[场景类型]中使用[道具系统]构建独特[策略体系]。

### 核心支柱分析框架
- **支柱 N：[具体体验]**
  - 体验：描述玩家感受到的价值
  - 设计手段：实现该体验的方法
  - 具体参数：数字化/规则化/集合化

### 枪火重生 (Gunfire Reborn) - 2026-03-04

**X-Statement**：国风第一人称射击游戏，融合 Roguelite 随机要素与 RPG 策略养成，玩家操控多位能力各异的英雄，在随机生成的关卡中使用种类繁多的武器与秘卷构建独特战斗流派。

**核心支柱**：
1. **国风题材 + 低多边形美术** - FPS Roguelite 中独树一帜的视觉风格
2. **英雄技能体系 + BD 构建** - 70+武器、200+秘卷、多位英雄带来的极高构筑策略深度
3. **多人合作** - 最多四人联机，社交玩法延长游戏生命周期

**玩法原型参考**：
- FPS + Roguelite 闯关：Immortal Redneck / Risk of Rain
- 武器系统：Borderlands 随机掉落
- 技能系统：Overwatch 英雄技能
- 秘卷系统：传统 Roguelike 随机增益

**基础信息**：
- 开发者：多益网络
- 品类：FPS + Roguelite + RPG
- 主题：国风（低多边形美术）
- 平台：PC (Steam) / 移动端

### 喵喵的结合 (Mewgenics) - 2026-03-05

**X-Statement**：喵星人回合制战术RPG，融合 Roguelite 随机冒险与遗传育种养成，玩家培育猫咪战队，通过交配繁殖将突变遗传给后代，在10x10战场中利用元素反应与环境互动构建独特战斗流派。

**核心支柱**：
1. **遗传育种系统** - 猫咪突变可遗传给后代，跨代际养成深度
2. **1000+技能/900+物品 BD构筑** - 12职业 × 75项专属 + 172项通用技能
3. **涌现式战斗设计** - 水+电传导、食物烹饪、环境互动

**特色机制**：
- 只有基础属性会遗传，突变和缺陷绝对遗传
- 元素反应：水+电传导、火+食物烹饪、湿润+电传导
- Heatwave机制：减少治疗效果，禁用战后自动恢复

**量化数据**：
- 12职业 / 1000+技能 / 900+物品 / 200+敌人
- 发售12天销量破100万
- PCU峰值：115,428人
- Steam好评率：97%

**与《以撒的结合》传承**：
- 旧酒：随机地牢 + 无限重玩 + 收集build
- 新瓶：遗传育种系统 + 涌现式元素反应

## 游戏数据分析方法（自我学习）

### Steam 游戏数据获取流程

1. **SteamDB** 获取用户活跃（峰值在线）
2. **SteamCharts** 获取历史玩家曲线
3. **搜索引擎** 获取媒体报道的玩家数据
4. **Steam 商店** 获取评价信息

**注意事项：**
- SteamDB 有 Cloudflare 防护，需用 Playwright 爬取
- Steam 商店评论需要登录，自动化会被屏蔽
- 游戏发售前可从测试/预购阶段获取预估数据
- 发售后从新闻报道获取首发数据
- 玩家评论可用搜索引擎从 Reddit/论坛获取

5. **赛季制长线运营** - "赛季回坑-赛季末流失"玩家生态

**与《无主之地》传承**：
- 旧酒：FPS射击 + 装备Build构筑 + 重复刷宝循环
- 新瓶：Raid团队协作 + 8年科幻叙事 + 赛季运营

**量化数据**：
- 10个 Raid / 10个地牢
- 3职业 × 6元素分支
- 8年运营，25+赛季
- Steam 历史峰值：316,651人（2023年2月）

**量化数据**：
- 6种 Runner / 6大派系 / 4+地图 / 6类武器Mod / 100+改装组合
- 测试期 PCU 峰值：143,621 人
