# 机制库

## 机制2：外观升级系统

| 字段 | 内容 |
| ----- | --------------------------------------------------------------------------------------------------------------- |
| 机制名称 | FPS-无畏契约-外观升级系统 |
| 核心标签 | 外观收集/附加付费/长线运营 |
| 一句话解释 | "通行证绑活跃+R点绑大R=全付费端覆盖" |
| 机制描述 | 皮肤可通过R点升级特效、动作、语音等外观部件；R点可通过对局或付费获取。核心洞察：把"活跃产出"变成"大R专属付费入口"。传统通行证绑活跃拉留存，Valorant让活跃产出R点，成为大R愿意持续付费升级皮肤的入口。 |
| 触发条件 | 玩家获得皮肤后，可消耗R点进行升级 |
| 解决痛点 | 传统皮肤一锤子买卖，付费用户后续无消费动力 |
| 体验目标 | 为大R提供持续付费理由，同时通过活跃产出R点绑定免费玩家活跃度 |
| 相似机制 | CSGO皮肤升级/Valorant战斗通行证 |
