# 机制库

## 机制1：技能系统

| 字段 | 内容 |
| ----- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| 机制名称 | FPS-无畏契约-技能系统 |
| 核心标签 | 技能组合/局内成长/FPS |
| 一句话解释 | "类似MOBA的技能+FPS枪法=策略深度" |
| 机制描述 | 每个英雄拥有独特技能组（1个终极技能+2个普通技能+1个被动技能），技能冷却独立计算。核心设计是降门槛：通过可视化的释放方式+技能包装的烟火闪，使得一个角色同时只需要干一件事。英雄技能包装的"烟火闪"，把旧酒《CSGO》需要复杂记忆的投掷物，变成了可视化、直接释放的英雄技能，极大降低门槛。 |
| 触发条件 | 回合开始前选择英雄，战斗中通过按键释放技能 |
| 解决痛点 | CSGO投掷物需记忆练习门槛高，守望先锋技能偏娱乐影响平衡 |
| 体验目标 | 降低战术道具门槛，同时保留战术深度，不破坏公平性 |
| 相似机制 | 守望先锋英雄技能/CSGO烟雾弹闪光弹 |
