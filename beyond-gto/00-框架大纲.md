## 核心框架（每场景四步）

每个场景独立成篇，每篇按以下结构展开：

1. **GTO/基线策略** — 标准打法是什么
2. **玩家池偏差** — 各类型玩家容易犯什么错
3. **剥削策略** — 基于错误用什么方案盈利
4. **反剥削风险** — 这个策略有哪些被反制的可能

---

## 玩家分级体系

```
Tier 0  鱼卵
        跟注站/疯鱼，小白，策略混乱
        → 识别：XXXXXX（待补充）
        → 策略：利用他们的犯错自动盈利，无需定制

Tier 1  鱼苗
        有范围概念，知道牌力大小，但翻前翻后策略都有显著漏洞
        以ABC紧弱玩家为主

Tier 2  小鱼
        有一定生存能力，靠捕食同级或其他低级鱼盈利
        Nit岩石玩家、松凶玩家
        漏洞和剥削策略同样容易找到

Tier 3  初级捕食者
        广义紧凶/TAG/Pro（线上平台最常见类型）
        20-25% VPIP × 12-16% PFR

        T3-1：翻前范围显著错误 / 诈唬不足
              例：10次3-bet showdown里全是QQ+，没有bluff
              或Kxx彩虹面进攻方本应高频Cbet却多次过牌

        T3-2：aggression太高，保护范围缺失
              例：深筹OOP在JT7听花面用KK大尺度连开三枪

        T3-3：翻后相对被动，过牌成瘾
              缺乏主动造池能力

Tier 4  顶级牌手
        基础行动线几乎无懈可击
        核心能力：识别对手池的集体偏移并针对性调整
        策略重心：防御优先
        博弈本质：二阶博弈——"我在想他认为我会怎么做"
```
