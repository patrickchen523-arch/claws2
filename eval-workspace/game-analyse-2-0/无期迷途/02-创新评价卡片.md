## 原型层判断

**品类原型本质上是一个「角色驱动的即时战术塔防+卡牌养成RPG」**，底层驱动力来自：
- 战斗层：手动操作+策略（而非自动战斗）
- 养成层：角色收集+数值养成
- 叙事层：主线剧情+角色个人剧情

对比同品类原型（如《明日方舟》），原型层特征高度一致：战前布阵→战中使用技能→阻挡敌人→击杀通关，角色稀有度抽卡+养成提升→继续推图。

**原型有变化（说明了什么变了）**：
- 战斗从半自动变为**全手动操作+移动换位**，改变了战斗的操作密度和策略维度
- 叙事从"配角式"变为**核心卖点之一**，审查系统让角色成为有情感的存在

## 结构创新

以下候选创新逐条通过3条硬性门槛：

### 1. BREAK核心击破机制

| 门槛 | 论证 |
|------|------|
| 可感知 | ✅ 每次BOSS战，玩家必须主动寻找核心、用核心伤害角色击破——击破瞬间有明显视觉反馈和战略转折 |
| 结构性 | ✅ 将"选对角色"从战力碾压变成策略必须，改变了"如何打败BOSS"的结构——不是更高伤害，而是特定机制触发 |
| 区分度 | ✅ 去掉BREAK机制，游戏退化为标准"放角色→等能量→放技能"循环，《无期迷途》与《明日方舟》的核心差异消失 |

**结论：通过门槛 → 结构创新**

### 2. 战中移动换位机制（限次操作）

| 门槛 | 论证 |
|------|------|
| 可感知 | ✅ 玩家在每关开始后可以拖动已上阵角色重新布阵，次数有限制——这是与所有主流塔防的核心差异 |
| 结构性 | ✅ 改变了战前准备的逻辑：不是一次性布阵就结束，而是"预判+实时调整"双层策略 |
| 区分度 | ✅ 去掉此机制，游戏退化为静态塔防布阵，《无期迷途》的手操深度优势消失 |

**结论：通过门槛 → 结构创新**

### 3. 审查/审讯角色互动系统

| 门槛 | 论证 |
|------|------|
| 可感知 | ✅ 玩家通过对话与角色互动，了解背景故事——这是抽卡后额外的情感投入，非强制但可感知 |
| 结构性 | ✅ 将"角色从战力变成有故事的人"，改变了角色收集的动机结构——不只是数值，也是情感认同 |
| 区分度 | ✅ 去掉审查系统，《无期迷途》退化为标准抽卡→养成→推图，角色和其他手游的差异消失 |

**结论：通过门槛 → 结构创新**

### 4. 无PVP纯内容向设计

| 门槛 | 论证 |
|------|------|
| 可感知 | ✅ 游戏中无任何竞技场/排行榜/对抗玩法，所有内容都是PVE验证——玩家从第一天就知道不会有"被人碾压"的挫败 |
| 结构性 | ✅ 改变了商业化逻辑：付费不能带来竞技优势，必须靠内容吸引（角色/剧情/美术） |
| 区分度 | ✅ 去掉无PVP设计，《无期迷途》变成标准有竞技场的二次元手游，与大多数竞品无异 |

**结论：通过门槛 → 结构创新**
