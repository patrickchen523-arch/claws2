## 一、游戏整体概述

**《杖剑传说》**（英文名：Journey of Candle and Sword / 简称杖剑）是由**雷霆游戏发行**的异世界轻冒险RPG手游。

- **画风**：日式清新手绘风格
- **世界观/舞台**：设定于"**坎斯汀世界**"
- **核心定位**：放置挂机快速升级 + 自走棋式回合战斗 + Roguelike技能搭配
- **类型标签**：放置类RPG / 异世界冒险 / 角色养成 / 回合制战斗
- **官方slogan**：「怪轻松的异世界RPG冒险手游，一起探索坎斯汀世界！」
- **TapTap评分**：8.6分
- **游戏融合**：日式清新手绘画风，兼顾轻松放置养成与自由探索战斗，挂机即可快速升级，主打自走棋式回合战斗 + 肉鸽技能搭配，支持组队秘境开荒

## 二、核心玩法/模式

### 1. 主线与日常
- **主线任务**：推进地图解锁功能（女神像、小推车、公会等），是获取经验和资源的最主要途径
- **每日玩法**：日常资源副本、伙伴养成、命运探索等

### 2. 六大轮换活动（循环活动）
| 活动名称 | 周期 | 特点 |
|---|---|---|
| **秘宝大作战** | 每周1次，每期7天，共5层 | 完成每日/循环任务获世界石，兑换5级奖励；零氪可"跳一期买两期"策略 |
| **菲涅克的谜题** | 三周1次 | 谜题由易到难，形式多样 |
| **宾果抽抽乐** | 三周1次 | 7天每日任务给28张奖券+命运探索补4张，需保证至少80苹果 |
| **幸运刮刮乐** | 三周1次 | 纯福利活动，建议囤素材秘境到活动期间使用 |
| **虚空裂隙** | 两周1次 | 建议组队（非伙伴），第5层难度较高 |
| **混沌侵袭** | 两周1次 | 打大怪加快召唤进度，BOSS出后组纯玩家队；晶块只换特殊遗物 |

### 3. 主题限时活动（春日版本"花海绮想"为例）
- **春野漫游**：7日签到，领宠物进阶素材、伙伴好感道具、限定家具"绮想花车"
- **幻装·春日游**：术士/战士两套花卉主题外观
- **趣味消消乐**：闯关模式（稳定产出）+ 竞速模式（排行榜），通关得繁花币、经验药水、强化石
- **拾花小铺**：用繁花币兑换伙伴礼物、限定花卉、主题外观
- **春日特供**：限定古遗物礼包、高性价比养成资源

### 4. 联动活动
- **刺客伍六七联动**：鸡大保等联动角色，获取方式与技能介绍
  - 伍六七：联动输出天花板，真实伤害+残血增伤，推图/打BOSS都强
  - 梅花十三：风系持续输出优选，稳定持续输出
  - 何大春：前排控制+破甲，强度一般
  - 鸡大保：签到7天免费送，范围提升友方会心率、范围治疗友方

### 5. 赛季制
- S4赛季进行中
- 预告"七转"职业进阶体系将开放，等级上限突破影响属性、技能形态、装备

## 三、关键系统/机制

| 系统 | 说明 |
|---|---|
| **职业体系** | 法师/战士/贤者/斗士/术士/骑士等，多转职路径；法师前期AOE强，贤者适合零氪，斗士/术士重氪强力 |
| **装备共鸣** | 根据全部已穿戴装备的强化等级触发共鸣，平均升级收益最高 |
| **古遗物** | 装备稀有道具，有属性加成；来源：副本、礼包、活动兑换；遗物属性分光/暗/水/风/火五元素 |
| **所属国度** | 森之国、山之国、泽之国、龙之国（乾坤造化/道衍天机）、羽之国（曙光秘誓）、其他 |
| **幻兽同调** | 特色养成系统，幻兽与角色联动 |
| **家园系统** | 资源产出（罗拉、木材等），有家园加成遗物 |
| **秘境副本** | 组队开荒，产出素材；可提前囤积用于活动期间消耗 |
| **圣兽试炼** | 累计参与天数提升防御等属性 |
| **祈愿系统** | 用来抽角色技能，开局会免费赠送技能抽数 |
| **称号系统** | 多种传说/奇迹品质称号，如"前有绝景""商人的青睐""虔诚的祈愿者"等 |
| **交易行** | 宝石、神铸石等物品的买卖省钱技巧 |

## 四、设计亮点

1. **放置+策略双轨**：挂机快速升级，同时需要研究职业搭配、技能组合、装备共鸣触发条件
2. **自走棋式回合战斗 + Roguelike技能**：战斗自动进行但技能可自由搭配，提供深度策略空间
3. **循环轮换活动结构**：6种轮换活动以不同周期滚动，玩家需要持续关注、合理规划资源
4. **零氪友好**：资源获取机制相对公平，策略可弥补战力差距，但养成周期漫长
5. **外观与联动**：持续推出主题外观和联动角色（刺客伍六七等），丰富付费维度
6. **多转职+稀有遗物BD**：职业多样，遗物词条丰富，提供多种Build方向
7. **护盾计算机制复杂**：固定值护盾、基于己方/目标属性的护盾、独立护盾各有结算规则
8. **联动内容深度整合**：联动角色（伍六七等）技能设计与原作角色性格高度契合
