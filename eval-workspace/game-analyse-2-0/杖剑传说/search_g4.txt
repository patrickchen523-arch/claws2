## 一、游戏整体概述

**杖剑传说**（英名未知）是一款由 **SSStudio** 开发的异世界轻冒险放置手游，**厦门雷霆网络科技股份有限公司** 发行运营。

- **公测时间：** 2025年5月29日
- **平台：** Android（适配 Android 7.0 及以上）
- **类型：** 竖屏放置RPG / 异世界轻松冒险社交MMORPG
- **世界观：** 玩家以冒险者身份穿越至"坎斯汀"（Kanstin）异世界大陆，拨开地图迷雾、收集珍稀宝物、与伙伴组队挑战圣兽

> 引用（开发者 TapTap 公告）："杖剑是一个怪轻松的异世界冒险放置游戏。在'轻松'的基调下，同时做了一个2D的可探索的，充满剑与魔法的异世界。"

---

## 二、核心玩法/模式

### 主要模式
1. **放置战斗（自走棋式回合战斗）** — 预设技能后由AI自动战斗，号称"炸裂输出引爆全场"
2. **大世界探索** — 2D可探索异世界，包含主线/支线剧情、遭遇战、解谜、开启宝箱等，为"大型线性箱庭关卡"
3. **角色养成与技能搭配（Roguelike）** — 百变技能自由搭配，打破职业刻板印象
4. **组队挑战** — 支持与NPC（原住民）组队，也支持真实玩家组队挑战PVE合作副本（圣兽）
5. **睡觉变强机制** — 放置即可成长，挂机快速升级

### 商业化定位
- **"降肝降氪"心智** — 通过降低肝度和氪金压力增强玩家留存
- **赛季制 + 拍卖行模式** — 计划延缓内容消耗和数值膨胀问题

## 三、关键系统/机制

| 系统 | 说明 |
|------|------|
| **技能系统** | 角色养成核心，千种技能搭配，Roguelike式随机技能组合 |
| **转职系统** | 职业自由切换，打破职业限制，减少练级负担 |
| **放置/睡觉成长** | 挂机即升级，挂空岛、看日出月落、边睡边变强 |
| **大世界探索** | 地图迷雾揭示、隐藏任务、未知宝藏 |

## 四、设计亮点（Design Highlights）

### 开发者设计理念（来源：TapTap 开发者的话）

开发者自述做游戏的**初衷与痛点解决**：

> "我本人是个游戏爱好者+泛二次元爱好者。喜欢各种二游的美术风格，但是不太喜欢千篇一律的养成和玩法。喜欢大地图探索的游戏乐趣，但是玩久了又肝不太动。想在游戏里认识盆友一起玩，但I人的性格在死命阻止我社交。因此，天空一声惊雷，杖剑传说应运而生~"

游戏试图解决的三大痛点：
1. **美术同质化** → 独特日式清新手绘画风
2. **肝度太高** → 竖屏放置 + 睡觉变强机制
3. **社交障碍** → 鼓励社交但不强制，支持NPC和真人组队

### 产品定位
- **"异世界轻松放置社交MMORPG"** — 多标签融合产品
- **竖屏设计** — 区别于传统重度手游，主打轻度休闲
- **自走棋式战斗 + Roguelike技能搭配** — 策略深度与轻度操作的平衡
- **"齿轮耦合"产出循环** — 将单点养成串联为互锁齿轮组，目标传递解决"下一步做什么"

### 开发历程（开发者自述）

| 时间节点 | 内容 |
|----------|------|
| 2019年底 | 开始构思，最早Demo为修仙题材（"仗剑长生"） |
| 2021年8月 | 正式立项 |
| 开发期间 | 题材更换、横屏→竖屏、完整替换美术资源、玩法反复迭代 |
| 公测前 | 大大小小十几次测试，无数次迭代 |
| 2025年5月29日 | 正式公测 |

> "杖剑成长至今，离不开一路陪伴、支持、鼓励、吐槽和鞭笞我们的可爱玩家们。万分感谢~"
