# 游戏分析报告：战棋品类

## 品类概述

战棋游戏（Turn-Based Strategy）是策略游戏的重要分支，以回合制战斗为核心，结合角色养成、地图策略等元素。本报告分析两款代表性战棋产品：XCOM 2（幽浮2）和 Divinity: Original Sin 2（神界原罪2）。

---

### 1.1 基础信息

| 字段 | 内容 |
| :--- | :--- |
| 游戏名称 | XCOM 2（幽浮2） |
| 开发商 | Firaxis Games |
| 上线时间 | 2016年2月5日 |
| 品类 | 战术策略战棋 |
| 玩家评价 | Steam 好评率 91% |
| 销量 | 累计销量超1000万份 |

### 1.2 X-Statement 公式

**基础信息** + **品类** + **核心差异点/独特卖点**

> 单人/多人（2-6人）回合制战术策略游戏，玩家指挥特种部队对抗外星人入侵，核心差异点是**高不确定性RNG + 掩体侧翼战术博弈**

### 1.3 玩法原型

- **玩法原型**：《XCOM 2》基于**不确定性策略博弈**的体验
- 借鉴了传统战棋的格子移动 + 掩体系统，但通过**极端的RNG机制**创造出独特的"痛并快乐着"体验
- 独特卖点：将战术决策与命运赌注结合，每次射击都是一次心理博弈

### 1.4 核心支柱

**支柱一：RNG不确定性系统**
- 体验：-> 设计手段：-> 具体参数
- 核心乐趣：-> 命中率随机波动（65%基础命中率，可被掩体、蹲伏闪避）-> 暴击率5%
- 设计意图：让每次攻击都充满悬念，即使是优势局面也可能翻盘

**支柱二：掩体与侧翼系统**
- 体验：-> 设计手段：-> 具体参数
- 全掩体：-40%受伤，-40%被命中率
- 半掩体：-20%受伤，-20%被命中率
- 侧翼攻击：取消掩体收益，额外+50%暴击率
- 设计意图：创造位置博弈，掩体是攻防转换的核心

**支柱三：行动点系统**
- 体验：-> 设计手段：-> 具体参数
- 每角色2行动点/回合
- 移动消耗1点，攻击消耗1点，蹲伏消耗1点
- 设计意图：资源有限性创造决策压力

**支柱四：物理破坏环境**
- 体验：-> 设计手段：-> 具体参数
- 可破坏掩体、木箱、油桶
- 油桶爆炸范围3格，伤害15-20
- 设计意图：场景互动增加战术维度

### 1.5 核心体验

**以秒为单位**：
> 3D等距视角，玩家指挥小队在网格地图移动，利用掩体隐藏，对敌人开火，享受"差一点就命中"的心跳时刻

**以分钟为单位**：
> 任务流程：潜入探索（2-3分钟）-> 遭遇战斗（5-15分钟）-> 撤离/失败，单局20-40分钟

## 二、Divinity: Original Sin 2（神界原罪2）分析

### 2.1 基础信息

| 字段 | 内容 |
| :--- | :--- |
| 游戏名称 | Divinity: Original Sin 2（神界原罪2） |
| 开发商 | Larian Studios |
| 上线时间 | 2017年9月14日 |
| 品类 | 开放世界战棋RPG |
| 玩家评价 | Steam 好评率 96% |
| 销量 | 累计销量超500万份 |

### 2.2 X-Statement 公式

**基础信息** + **品类** + **核心差异点/独特卖点**

> 1-4人合作回合制战棋RPG，玩家扮演逃犯在魔法世界冒险，核心差异点是**元素环境互动系统 + 极致自由的角色构建**

### 2.3 玩法原型

- **玩法原型**：《神界原罪2》基于**桌面RPG + 元素互动**的体验
- 借鉴了D&D（龙与地下城）的自由角色创建 + 技能组合
- 独特卖点：**DIP系统**（元素地板+召唤物+投射物组合），让战斗变成化学实验

### 2.4 核心支柱

**支柱一：元素互动系统**
- 体验：-> 设计手段：-> 具体参数
- 6种基础元素：水、火、冰、电、毒、土
- 元素组合：火+油=爆炸，冰+水=结冰，电+水=导电
- 设计意图：让环境成为战略资源，鼓励玩家利用地形

**支柱二：DIP组合系统**
- 体验：-> 设计手段：-> 具体参数
- D=地板元素技能（如火焰领域）
- I=召唤元素生物（如火焰精灵）
- P=投射物附加元素（如火箭矢）
- 组合效果：地板元素可被召唤物继承，投射物可沾染地板元素
- 设计意图：创造无限可能的技能combo

**支柱三：极致自由角色构建**
- 体验：-> 设计手段：-> 具体参数
- 14个种族 + 6个初始职业
- 技能树：每个职业有30+技能
- 属性点：力量、智力、敏捷、体质、智慧
- 设计意图：每个玩家都能创造独特的角色build

**支柱四：4人合作战斗**
- 体验：-> 设计手段：-> 具体参数
- 技能AP共享机制
- 位置协作奖励
- 敌人AI会优先攻击最近单位
- 设计意图：合作创造协同效应

### 2.5 核心体验

**以秒为单位**：
> 3D等距视角，玩家在开放世界探索，与NPC对话或战斗，战斗时利用元素组合秒杀敌人，享受"发现新combo"的惊喜

**以分钟为单位**：
> 探索阶段（15-30分钟）-> 遭遇战斗（10-20分钟）-> 战斗胜利/失败 -> 剧情选择，单局体验差异大

## 三、战棋品类设计洞察

### 3.1 核心设计模式

| 模式 | XCOM 2 | 神界原罪2 |
| :--- | :--- | :--- |
| 策略深度 | 位置博弈（掩体侧翼） | 元素组合（化学反应） |
| 不确定性 | 高（RNG命中小数） | 中（骰子判定较温和） |
| 角色成长 | 线性（兵种解锁） | 自由（技能组合） |
| 环境互动 | 破坏（油桶） | 元素（创造陷阱） |

### 3.2 设计启示

1. **不确定性是战棋的灵魂**：XCOM的极端RNG创造了"心跳体验"，但需要配合足够的策略深度让玩家"虽败犹荣"

2. **环境互动提升战术维度**：从简单的可破坏物到元素组合，环境应成为战略资源

3. **角色构建自由度的取舍**：神界原罪2的高自由度适合核心玩家，XCOM的线性成长适合快速上手

4. **合作模式的价值**：4人合作战棋是蓝海市场，协同战斗体验独特
