# 主玩法：回合制对战

## 基本信息

| 字段 | 内容 |
| :--- | :--- |
| 玩法名称 | 回合制对战 |
| 所属游戏 | 洛克王国：世界 |
| 玩法类型 | 主玩法（核心循环） |
| 核心规则 | 回合制策略对战，魔力值+能量管理+属性克制 |
| 分析日期 | 2026-04-10 |

---

## 核心循环

```
选择精灵入场 → 每回合选择技能/防御/切换/道具
  → 属性克制×能量管理 → 战斗胜利
  → 获得洛克贝/经验/碎片 → 养更强精灵
  → 挑战更强对手 → 循环
```

---

## 目标

- 击败对手（NPC/BOSS/玩家）
- 提升天梯段位
- 验证养成成果

---

## 约束

- **魔力值限制**：双方各有3点魔力，被击败失去1点，魔力耗尽战斗失败
- **能量点限制**：所有技能共享能量池，高威力高消耗
- **场地天气**（仅PVE）：天气/场景对特定精灵产生增益或减益

---

## 节奏

- 普通遭遇：5-15分钟
- 天梯排位：15-20分钟
- BOSS战：多阶段（先破盾再打本体）

---

## 涉及机制

- 能量池替代PP值（核心资源管理）
- 18属性克制体系
- 场地相性PVE
- 精灵血脉技能（被动）
- 辅助宠物回能技能

---

## 体验画像

**典型体验曲线**：
- 开局：选择精灵+思考能量分配
- 中局：属性博弈+能量紧张→切换宠物
- 结尾：决胜回合或被翻盘

**节奏类型**：思考型节奏，有回合间隔供决策

---

## 设计启示

回合制对战的策略深度来自两个维度：
1. **能量分配**：每回合决策"能量怎么花最值"
2. **属性博弈**：属性克制链×场地天气三维影响

两者叠加形成丰富策略空间，但上手门槛低（无命中/暴击简化运气因素）。
