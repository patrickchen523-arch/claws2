# 副玩法：开放世界探索

## 基本信息

| 字段 | 内容 |
| :--- | :--- |
| 玩法名称 | 开放世界探索 |
| 所属游戏 | 洛克王国：世界 |
| 玩法类型 | 副玩法（内容供给型） |
| 与主玩法关系 | 独立型 |
| 分析日期 | 2026-04-10 |

---

## 核心循环

```
进入栖息地 → 观察精灵生态行为
  → 触发遭遇或发现隐藏内容
  → 捕捉精灵/获取道具
  → 放入宠物园/出战队伍
```

---

## 一句话定位

无缝大世界自由探索，发现精灵生态行为和隐藏内容。

---

## 体验差异

| 维度 | 主玩法（回合制对战） | 副玩法（开放世界探索） |
|------|-------------------|---------------------|
| 核心目标 | 击败对手 | 发现精灵/NPC/宝箱 |
| 操作方式 | 回合制策略选择 | 无缝大世界自由移动 |
| 胜负反馈 | 胜/负/逃跑 | 发现物数量/隐藏内容 |
| 时间节奏 | 回合节奏，可暂停思考 | 连续时间流，不可暂停 |
| 典型场景 | 对战竞技场/BOSS | 大地图野外/生态栖息地 |

---

## 设计意图

让精灵"活"在世界中，而非静态图鉴。天气/昼夜影响精灵刷新率，提升真实感和探索策略性。

---

## 支撑机制

- 精灵生态AI系统（核心）
- 动态昼夜+天气系统
- 隐藏宝箱/NPC/解谜事件
- 骑乘/飞行探索
- 特定地形需要特定精灵能力

---

## 节奏曲线

- 探索期：连续移动→观察生态→发现稀有精灵
- 遭遇期：触发战斗或捕捉
- 收获期：获得新精灵或道具

---

## 设计启示

开放世界探索为回合制对战提供了"前戏"——玩家在大世界中发现、观察、捕捉精灵的过程本身是乐趣，而不只是为了"抓更强的怪来打"。
