# 游戏机制库

> 这里是俊亦的游戏机制知识库，持续收集和整理各种游戏机制设计。

## 目录结构

```
mechanism-lib/
├── core-systems/      # 核心系统机制
│   ├── combat/        # 战斗系统
│   ├── progression/   # 成长系统
│   ├── economy/       # 经济系统
│   └── social/        # 社交系统
├── genre-mechanics/  # 品类特定机制
│   ├── roguelike/
│   ├── fps/
│   ├── moba/
│   └── tcg/
├── design-patterns/  # 设计模式
│   ├── feedback/
│   ├── difficulty/
│   └── engagement/
└── references/       # 参考资料
```

- 查找特定机制：告诉机制君你需要什么类型的机制
- 添加新机制：提供机制名称、来源游戏、核心原理
