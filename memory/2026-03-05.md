# Daily Memory - 2026-03-05

## Ark Nova 机制复盘

### 成功入库：机制一 + 机制二

**机制一：双轨竞争系统（双轨竞争/资源权衡）**
- **为什么好**：
  1. 两条轨道相互嵌套制约，提升一个需要消耗另一个的资源，打破"刷到顶"的单调感
  2. 差值决定胜负的设计让"平衡发展"成为策略核心，而非单一最优解
  3. 视觉上轨道嵌套，机制与主题（动物园既要吸引游客又要保护动物）高度统一

**机制二：卡牌位置决定行动强度（行动选择/手牌管理）**
- **为什么好**：
  1. 用"位置"代替复杂的数值计算，简化决策复杂度
  2. 创造"顺序规划"策略维度——什么时候打什么牌变得重要
  3. 借鉴自《Civilization: A New Dawn》，已有成功先例
  4. 每次打牌后卡牌重排，动态变化增加策略变数

### 未入库：机制三、四、五

**机制三：动物卡牌引擎** - 太像《Wingspan》，没有独特创新点
**机制四：保护项目计分** - 条件计分是常见设计，缺乏独特性
**机制五：赞助商系统** - 被动收益也是常见机制，缺乏差异化

### 总结：什么样的桌游机制值得入库？

1. **机制与主题高度统一**（双轨=动物园双重使命）
2. **有独特的策略维度**（卡位强度=顺序规划）
3. **机制足够抽象可复用**（不是针对特定游戏的数值设计）
4. **有借鉴先例但有改进**（借鉴Civilization但做了适配）
