# 2026-03-09 工作日志

## 8:00 - 每小时游戏拆解任务

### 分析对象：蛋仔派对（派对游戏品类）

**关键发现**：
- 玩法原型：借鉴《糖豆人》Fall Guys的多人闯关派对玩法
- 核心差异：加入UGC地图工坊（核心竞争力）+ 本土化社交功能
- 成绩：2022年上线，累计5亿注册用户，春节日活4000万+

### 入库内容
- 游戏卡：1款（蛋仔派对）
- 机制卡：4个
  - UGC地图工坊：解决派对游戏内容消耗问题
  - 道具对抗系统：增加综艺效果和意外变量
  - 魂蛋观战模式：淘汰玩家可继续干扰对手
  - 变异蛋狂潮：Roguelike割草单人玩法

### 学到的
1. 派对游戏核心竞争力=内容消耗速度，UGC是解法
2. 魂蛋模式是本土化创新（原版糖豆人没有）
3. "综艺效果"设计要点：意外性、戏剧性、社交话题性
