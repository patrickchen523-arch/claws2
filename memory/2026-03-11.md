# 2026-03-11 工作日志

## 新项目：宝可梦搜打撤游戏原型

### 游戏概念
- **类型**：单人回合制PVE搜打撤
- **核心**：宝可梦战斗系统 + 体力条限制
- **参考**：宝可梦战斗、三角洲的赚钱、梦幻合宠

### 核心循环
1. **战斗内**（秒级）：宝可梦选招回合制战斗、多精灵切怪
2. **单局**（分钟级）：体力值限制，环形地宫，越深战利品越丰富
3. **长线**：弱数值系统，宠易满数值但有方向分化和打书深度

### 经济循环
- 捉宠出售（对标 Pokemmo）
- 治疗精灵花销（对标搜打撤修装备）
- 合宠养宠（对标梦幻）
- 消耗道具

### Gemini 建议的 MVP 模块

**1. 核心战斗**
- 3种互相克制的属性（水、火、草）
- 预设3-4只精灵，固定技能
- 回合制逻辑、血条计算、属性克制、切怪功能

**2. 体力驱动的地宫探索**
- 3层环形节点地图（外层、中层、核心层）
- 移动和战斗回合都扣除体力
- 外层安全/低级怪 → 核心层精英怪
- 撤离点设定

**3. 单局结算**
- 背包容量限制
- 死亡/体力耗尽 = 战利品清零
- 战利品→金币→治疗消耗

### 待验证的核心体验
玩家在中层时，看着剩余体力，选择继续深入还是撤退的"患得患失"

### 讨论结论（确定版）

**1. 体力系统**
- 默认体力：100
- 移动消耗：1体力/格
- 战斗消耗：遇敌固定扣4点 + 每回合扣1点
- 体力0~-24：每消耗1体力额外扣除4%最大血量
- 体力≤-25：判定撤离失败
- 逃跑扣除：40体力

**2. 地图系统**
- 双层结构：外层Node Map + 内层8x8矩阵
- 明雷：任务点/BOSS
- 暗雷：草地/水路，概率遇怪
- 地形：空地、门、草地、水路、战利品点

**3. 背包系统**
- 3x4 = 12格，1格=1物品（保留size属性）

**4. 战斗系统**
- 传统回合制
- 切宠当回合不能攻击

**5. 难度梯度**
- 外层/中层/核心，数值递增

**6. 初始阵容**
- 3只基础宠

**7. 经济**
- 战利品：白色1块，每稀有度×5
- 治疗：早期固定5元/次

## PRD 已接收 (2026-03-11)

### 核心概述
- 2D俯视角地图探索 + 回合制RPG + 搜打撤经济
- MVP目标：验证体力值限制下的地宫探索与回合制战斗

### 体力系统
- 初始100点
- 移动：-1/格
- 遇敌：-4点
- 战斗回合：-1/回合
- 逃跑/战败：-40点
- 0~-24：透支惩罚（每点体力扣4%最大HP）
- ≤-25：失败，战利品全丢

### 地图系统
- 宏观：Node Map，撤离点间隔≤5
- 微观：8x8矩阵
- 地形：Empty, Door, Grass, Water, Loot, Visible_Enemy

### 战斗系统
- 速度轴排序
- 技能/切宠/道具都消耗回合
- MVP：本地预设5-10套镜像配置

### 背包
- 3x4 = 12格
- 1格1物品，预留width/height属性

### 数据结构
- Item: item_id, name, type, width, height, value
- Pet: pet_id, name, element, max_hp, current_hp, speed, skills

### 属性克制


### 技能库（每宠2技能）
- 普攻：无冷却
- 特技：1回合冷却

### 首批3只测试精灵
1. 火蜥蜴 - 火系，速度120，高攻脆皮
2. 水龟 - 水系，速度80，高血坦克，带回血
3. 草籽 - 草系，速度100，均衡型，吸血续航

### 后续机制（MVP后开发）

**受伤残值系统**
- 目的：防止切治疗宠无限制回血
- 机制：
  - 受伤残值 = 本次对局可治疗的最大HP
  - 初始 = 进入地图时的HP
  - 每次受伤后，受伤残值减少本次伤害的20%
  - 例：满血100被打到1，受伤残值减少约20，最多只能回到80%
