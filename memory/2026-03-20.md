## 踩坑记录

### Web 资讯抓取失败（游戏葡萄）

**问题**：游戏葡萄（games indices）网站使用 Strikingly 平台构建，Cookie 弹窗阻挡了所有自动化抓取工具。

**尝试过的方法**：
1. Jina Reader（`https://r.jina.ai/[URL]`）→ 失败，返回"需要 Cookie 同意"
2. 直接 `web_fetch` → 失败，页面内容为空
3. `web_search` 引擎缓存 → 失败，内容不完整
4. `browser` 工具自动化点击 Cookie 按钮 → **成功**（`focus` → `snapshot` → `act:click`），但等待时间较长

**结论**：Strikingly 平台建议直接用浏览器自动化处理，或换用其他无弹窗的数据源。
