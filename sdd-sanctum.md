## 3. 玩法原型

### 3.1 核心借鉴/体验原型
- **FPS原型**：Quake系列（武器手感：Tesla=闪电枪，Alt-fire=轨道炮）
- **塔防原型**：传统TD（Plants vs. Zombies、Orcs Must Die）
- **创新点**：**迷宫建造**——玩家不是简单放置塔，而是用基础方块塔构建自己的地图路径

### 3.2 核心设计创新
- 敌人会**自动寻路**：如果到核心的路径被阻挡，会自动选择下一条最快路线
- 玩家通过建造迷宫**主动规划敌人路线**，延长敌人行军时间
- **双重身份**：塔防建造者 + FPS战士

## 4. 核心支柱（4个）

### 支柱一：【迷宫建造与路径规划】
- **支柱体验**：通过建造迷宫主动引导敌人行军路线，最大化敌人暴露在火力下的时间
- **设计手段**：
  - 基础方块塔必须放置后才能在其他塔上建造炮塔
  - 敌人AI自动寻路，会绕过障碍选择最短路径
  - 地图存在多条可行路径，玩家可封锁捷径
- **具体参数**：
  - 塔可升级5级（每级增强伤害/射速/范围）
  - 不同地图有不同地形（双入口、飞行敌人等）
  - 资源有限，需精确规划塔的类型与位置

### 支柱二：【FPS战斗参与】
- **支柱体验**：玩家不仅是观察者，更是战场参与者，直接射击敌人
- **设计手段**：
  - 回合开始后可手动射击敌人
  - 战前选择武器配置（主武器+副武器）
  - 武器有Alt-fire（副火）机制
- **具体参数**：
  - 武器类型：突击步枪、霰弹枪、Tesla（闪电）、冻结枪、狙击枪
  - Tesla满级后伤害极高，可秒杀敌人
  - 霰弹枪Alt-fire可一次装填多发，伤害翻倍

### 支柱三：【4人合作分工】
- **支柱体验**：团队协作才能应对高难度波次
- **设计手段**：
  - 最多4人联机
  - 每人可独立建造塔和选择武器
  - 敌人波次数量和强度随难度递增
- **具体参数**：
  - 4种难度：Easy、Normal、Hard、Nightmare
  - 资源有限，团队需分工（有人负责建造，有人负责射击）

### 支柱四：【成长与资源管理】
- **支柱体验**：通过积累资源解锁更强力的装备，逐步变强
- **设计手段**：
  - 击杀敌人获得资源与经验值
  - 资源用于建造新塔或升级现有塔
  - 解锁新武器、塔类型和技能
- **具体参数**：
  - 每座塔可升级5次
  - 每种武器可升级5次
  - 87个Steam成就

## 5. 核心体验

### 5.1 以秒为单位的体验
| 时刻 | 体验描述 |
|------|----------|
| **0-5秒** | 回合开始，敌人从spawn点涌出，玩家开始射击 |
| **5-30秒** | 第一批敌人接近第一座塔，开始遭受交叉火力 |
| **30-60秒** | 玩家在关键路口补塔或切换武器射击漏网之鱼 |
| **60-90秒** | 迷宫深处传来枪声和爆炸声，第一批敌人抵达核心 |
| **90-120秒** | 波次结束，结算资源，开始下一回合规划 |

### 5.2 以分钟为单位的体验
| 时长 | 体验描述 |
|------|----------|
| **5-10分钟** | 单张地图1-3波次，逐渐理解敌人路线，开始规划迷宫 |
| **15-20分钟** | 某张地图第一次完成完美迷宫设计，敌人绕远路时被全灭的爽感 |
| **30-45分钟** | 经历多次失败后终于通关一张高难度地图 |
| **2-3小时** | 通关所有地图，体验不同难度下的策略变化 |

### 5.3 以小时为单位的体验
| 时长 | 体验描述 |
|------|----------|
| **5-8小时** | 完成单人模式大部分内容，理解游戏机制 |
| **10-15小时** | 尝试不同塔组合、武器搭配，寻找最优解 |
| **15+小时** | 挑战高难度，与好友联机合作，体验完整游戏内容 |

## 6. 总结

Sanctum作为**世界上首批FPS+塔防混合游戏**，核心创新在于：
1. **迷宫建造系统**——不是单纯放置塔，而是主动设计敌人路线
2. **双重身份**——既是塔防设计师，又是FPS战士
3. **硬核难度**——资源极度有限，容错率低，需要深度策略思考

> "Definitely one of the best tower-defense experiences out there." — Reddit评价

**主要批评**：难度过高，单人体验不如多人合作有趣，FPS部分相对鸡肋。
