# 品类原型词典（Genre Archetype Dictionary）

本文件列出主流游戏品类原型及其标志特征，用于快速匹配判断。
分析游戏时，优先从本词典中匹配最接近的原型，再判断差异。

---

## SLG 类

### COK-like（城堡争夺型 SLG）
- **标志特征**：个人城池建造 + 大地图资源点争夺 + 联盟协作攻城 + 迁城机制
- **核心循环**：建城发育 → 练兵升级 → 联盟集结 → 攻城掠地 → 赛季/服务器重置
- **乐趣来源**：成长碾压、联盟社交、攻城对抗
- **代表产品**：Clash of Kings、万国觉醒（Rise of Kingdoms）、无尽冬日（Last War）、King of Avalon
- **常见新瓶**：末日、丧尸、中世纪、三国、冰雪末世

### 率土-like（地缘博弈型 SLG）
- **标志特征**：大地图格子占领 + 行军真实耗时 + 武将配队策略 + 赛季制 + 强联盟政治博弈
- **核心循环**：占地扩张 → 武将配队 → 联盟博弈/外交 → 赛季争夺 → 赛季轮替
- **乐趣来源**：地缘博弈、策略配队、联盟政治、赛季竞技
- **代表产品**：率土之滨、三国志·战略版、鸿图之下
- **与COK-like的区别**：更强调地缘和博弈深度，弱化个人城池建造，赛季制更核心

### 4X-like（探索扩张型 SLG）
- **标志特征**：Explore-Expand-Exploit-Exterminate + 科技树 + 文明发展 + 回合/半即时制
- **核心循环**：探索版图 → 扩张领土 → 开发科技/经济 → 征服对手
- **乐趣来源**：文明成长、策略规划、科技路线选择
- **代表产品**：文明（Civilization）、无尽的拉格朗日、Stellaris

## RPG / 卡牌类

### 卡牌RPG-like（抽卡养成型）
- **标志特征**：抽卡获取角色 + 角色养成（升级/突破/装备）+ 配队策略 + 关卡推进
- **核心循环**：抽卡 → 养成 → 配队 → 推关/消耗内容 → 等版本更新 → 再抽卡
- **乐趣来源**：收集角色、养成成长、配队策略、角色情感投入
- **代表产品**：原神、崩坏：星穹铁道、明日方舟、阴阳师、FGO
- **常见新瓶**：二次元、三国、西游、末日、奇幻

### 回合制RPG-like
- **标志特征**：回合制战斗 + 数值养成 + 剧情驱动 + 单机或弱联网
- **核心循环**：推剧情 → 战斗 → 升级 → 推更难剧情
- **乐趣来源**：剧情沉浸、策略战斗、角色成长
- **代表产品**：梦幻西游、勇者斗恶龙、最终幻想（早期）

### ARPG-like（动作角色扮演型）
- **标志特征**：实时操作战斗 + 角色养成 + 装备/技能Build + 关卡或开放世界
- **核心循环**：刷怪 → 掉落装备 → 强化Build → 刷更难内容
- **乐趣来源**：操作爽感、Build构筑、装备掉落惊喜
- **代表产品**：暗黑破坏神、流放之路、火炬之光

## MMO 类

### 传统MMO-like
- **标志特征**：大世界 + 职业体系 + 公会 + 副本/团本 + 装备追求 + 长线身份成长
- **核心循环**：做日常 → 打副本 → 追装备 → 提升战力 → 打更难副本 → 社交/公会活动
- **乐趣来源**：社交身份、装备追求、团队协作、长线目标
- **代表产品**：魔兽世界、剑网3、逆水寒、天涯明月刀
- **常见新瓶**：武侠、仙侠、西幻、科幻

### 轻量MMO-like（社交驱动型）
- **标志特征**：弱数值 + 强社交/外观 + 开放世界探索 + UGC内容
- **核心循环**：社交互动 → 外观收集 → 探索 → 表达/创作
- **乐趣来源**：社交关系、外观表达、世界探索
- **代表产品**：光·遇、摩尔庄园、蛋仔派对（部分）

## 放置 / Idle 类

### 经典放置-like
- **标志特征**：离线收益 + 自动战斗 + 阶段性突破卡点 + 低操作频率
- **核心循环**：挂机积累 → 突破卡点 → 解锁新内容 → 继续挂机
- **乐趣来源**：数字成长、卡点突破的爽感、碎片时间友好
- **代表产品**：剑与远征、不思议迷宫、Idle Heroes

### 放置+X混合型
- **标志特征**：放置核心 + 叠加SLG/RPG/模拟经营外壳
- **核心循环**：放置积累 + 外围系统操作（建造/对抗/社交）
- **乐趣来源**：放置的轻松感 + 外围系统的目标感
- **代表产品**：无尽冬日（放置+COK）、咸鱼之王（放置+卡牌）

## 射击 / 竞技类

### BR-like（大逃杀型）
- **标志特征**：多人同场 + 搜刮装备 + 缩圈 + 最后生存者胜
- **核心循环**：跳伞 → 搜装备 → 交战 → 缩圈 → 决赛圈 → 再开一局
- **乐趣来源**：紧张感、操作对抗、随机性、"吃鸡"的成就感
- **代表产品**：PUBG、和平精英、Fortnite、Apex Legends

### 战术射击-like
- **标志特征**：小队对抗 + 角色/技能差异化 + 回合/攻防制 + 高致死率
- **核心循环**：匹配 → 选角/选装 → 攻防对抗 → 排名变化 → 再开
- **乐趣来源**：团队配合、策略执行、操作精度
- **代表产品**：Valorant、CS2、Rainbow Six

### MOBA-like
- **标志特征**：5v5 + 英雄选择 + 三路推塔 + 局内成长 + 团战
- **核心循环**：匹配 → 选英雄 → 对线发育 → 团战 → 推基地 → 再开
- **乐趣来源**：英雄操作、团队配合、排位竞技
- **代表产品**：王者荣耀、英雄联盟、DOTA2

## 模拟经营 / 建造类

### 农场经营-like
- **标志特征**：种植/养殖 + 收获/加工 + 交易/社交 + 装饰/表达
- **核心循环**：种植 → 收获 → 加工 → 卖出/交换 → 扩建 → 种更多
- **乐趣来源**：经营优化、装饰表达、社交互动、治愈感
- **代表产品**：Hay Day、星露谷物语、小森生活

### 城市建造-like
- **标志特征**：规划布局 + 资源管理 + 人口/交通/产业链 + 效率优化
- **核心循环**：建造 → 管理资源 → 优化布局 → 扩张 → 解决新问题
- **乐趣来源**：规划成就感、优化乐趣、视觉成长
- **代表产品**：Cities: Skylines、模拟城市、江南百景图

### 生存建造-like
- **标志特征**：资源采集 + 建造庇护所 + 环境威胁 + 探索 + 制作系统
- **核心循环**：采集 → 建造 → 探索 → 对抗威胁 → 升级基地 → 再探索
- **乐趣来源**：生存压力、探索发现、建造成就、自由度
- **代表产品**：Minecraft、Valheim、帕鲁（Palworld）、七日杀

---

## Roguelike / Roguelite 类

### 经典Roguelike-like
- **标志特征**：随机生成关卡 + 永久死亡（或部分继承）+ 局内Build构筑 + 高重玩性
- **核心循环**：进入地牢 → 随机选择升级/道具 → 推进 → 死亡/通关 → 局外成长 → 再进
- **乐趣来源**：随机性带来的新鲜感、Build构筑乐趣、挑战突破
- **代表产品**：Hades、杀戮尖塔、以撒的结合、弹壳特攻队

---

## 塔防类

### 经典塔防-like
- **标志特征**：路径防守 + 塔/单位放置 + 波次进攻 + 资源管理
- **核心循环**：布阵 → 迎敌 → 调整策略 → 过关 → 新关卡
- **乐趣来源**：策略布阵、难度挑战、通关成就
- **代表产品**：Kingdom Rush、保卫萝卜、明日方舟（塔防+卡牌RPG混合）

---

## 派对 / 休闲类

### 派对竞技-like
- **标志特征**：多人同场 + 短局 + 简单操作 + 随机关卡 + 淘汰/积分制
- **核心循环**：匹配 → 玩小游戏 → 淘汰/积分 → 再来一局
- **乐趣来源**：社交欢乐、低门槛竞技、随机性
- **代表产品**：糖豆人、蛋仔派对、Among Us

### 三消-like（Match-3）
- **标志特征**：网格消除 + 关卡制 + 道具辅助 + 外围Meta系统（装修/剧情等）
- **核心循环**：进关 → 消除 → 过关 → 推进Meta → 进更难关
- **乐趣来源**：消除爽感、闯关成就、Meta系统的目标驱动
- **代表产品**：Candy Crush、梦幻花园、Royal Match

### 混合休闲-like（Hybrid Casual）
- **标志特征**：核心玩法极简（合并/消除/跑酷等）+ 外围Meta系统（养成/建造/剧情）+ 强广告变现
- **核心循环**：核心操作 → 获得奖励 → 推进Meta → 再操作
- **乐趣来源**：核心操作的即时满足 + Meta系统的长线目标
- **代表产品**：Top War、Merge Mansion、Archero

## 使用说明

1. 分析游戏时，先在本词典中找到**最接近的母体**
2. 如果完全匹配，直接引用母体名称（如"COK-like"）
3. 如果是两个母体的混合，写成"A+B混合型"（如"放置+COK混合型"）
4. 如果本词典中没有匹配项，说明可能是新母体或极小众品类，需要在报告中详细说明
5. 本词典会持续更新，发现新的成熟母体时应补充进来
