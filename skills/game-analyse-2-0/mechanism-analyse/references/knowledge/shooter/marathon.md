# Marathon 机制知识卡

**品类原型**：搜打撤（Extraction Shooter）
**创新定性**：原型+结构创新
**分析日期**：2026-03-19（更新）

---

## 结构创新

### 动态撤离点
- **来源子模块**：关卡系统 6.3 探索要素
- **做法**：每局撤离点半随机刷新位置（不在固定位置）；激活后1分钟窗口；单人撤离成功后点位从地图消失
- **解决的问题**：解决了"蹲撤离点导致博弈僵化"的品类痛点——如果撤离点固定，高端玩家会提前埋伏，普通玩家体验极差
- **协同关系**：链式—动态事件改变路线 → 交战升级 → 撤离点博弈形成高压结算
- **移植注意**：需要足够清晰的撤离信息提示UI，否则新手迷失感强

### 噪音暴露系统
- **来源子模块**：战斗系统 1.7 战术层
- **做法**：移动/搜刮/战斗/开门产生不同级别噪音（低/中/高），地图上实时显示噪音指示器，敌人和AI都能响应噪音位置
- **解决的问题**：解决了"搜打撤信息战太被动"的问题——让玩家主动权衡"制造噪音获取优势"vs"保持安静但失去先手"，将信息获取从被动搜索变成主动决策
- **协同关系**：乘法—噪音暴露 × 动态撤离点 = 空间未知 × 信息未知的双未知博弈核心
- **移植注意**：必须有清晰的噪音UI指示器；潜行玩法需要相应Buff（如消音武器/静步技能）作为对冲

### 烟雾克制AI仇恨
- **来源子模块**：战斗系统 1.5 状态效果
- **做法**：烟雾弹释放后完全消除UESC巡逻队仇恨，可穿越危险区域而不触发战斗
- **解决的问题**：解决了"AI威胁无法被策略性管理"的痛点——让PvE威胁不是纯粹的风险，而是可以通过战术道具管理的变量
- **协同关系**：弥补—烟雾 × 职业系统 = 不同职业利用烟雾的能力差异形成战术互补（治疗职业穿越危险区最有价值）
- **移植注意**：烟雾必须同时遮挡玩家视野（PvP惩罚），否则会变成纯收益无代价的道具

### 三层库存分层
- **来源子模块**：经济系统 3.6 库存管理
- **做法**：局内背包（携带中，全损风险）→安全箱（局内保护，死亡不丢失，有容量上限）→局外仓库（永久保存）
- **解决的问题**：解决了"风险收益缺乏梯度"的品类痛点——让保守玩家和激进玩家都能找到适合自己的策略
- **协同关系**：乘法—三层库存 × 装备全损 = 清晰风险梯度让每件物品的存放位置都成为有意义的策略决策
- **移植注意**：安全箱容量必须限制（不能太大），才能真正产生取舍压力

## 设计精华

> Marathon 成立的关键在于构建了一个"双未知+有策略"的撤离博弈——空间未知（动态撤离点）× 信息未知（噪音暴露）× 风险有梯度（三层库存）。这三者共同让每局撤离都是真正的决策，而非"到固定地点赌一把"。

## 体验支柱

| 支柱 | 支撑机制 | 核心协同 |
|------|---------|---------|
| 双未知撤离博弈 | 动态撤离点 + 噪音暴露系统 | 乘法：空间未知 × 信息未知 = 每局都是真正的探索决策 |
| 战术配合深度 | Runner Shell职业 + 烟雾克制AI + Proximity语音 | 弥补：职业差异 × 烟雾 = 功能互补的组队策略 |
| 叙事合理化死亡 | 克隆体设定 + 永久进度 + Raid终局叙事 | 保险丝：克隆体降低死亡心理门槛 × 永久进度保证回来仍有事做 |
| 高风险价值回收 | 三层库存 + 装备全损 | 乘法：风险分层 × 收益梯度 = 策略分化最大化 |

## 承重墙机制

| 机制 | 拿掉后的影响 |
|------|------------|
| 噪音暴露系统 | 去掉后遭遇博弈退化为标准FPS蹲点，核心差异化消失 |
| 动态撤离点 | 去掉后博弈退化为塔科夫式蹲守，品类差异化消失 |
| 三层库存分层 | 去掉后风险收益变成"全有或全无"，策略分化消失 |
