# game-report-generator 经验总结

## 技能简介
生成游戏分析报告的 HTML/PDF 文档。当需要将 game-analyse-sdd 的分析输出转换为可视化报告时使用此技能。

## 使用经验

### 1. 工作流程

```
收集分析数据 → 生成 HTML 预览 → 浏览器查看 → 打印为 PDF
```

### 2. 报告结构

**第一页**：
- 游戏产品名称
- X-Statement
- 玩法原型
- 游戏支柱

**第二页**：
- 核心体验（+ 循环图）
- 目标用户
- 竞争环境
- 其他补充信息

### 3. 可视化程度

**当前实现**：
- 游戏循环：文字 + 箭头
- 支柱实现：精简短语

**待改进方向**：
- 嵌入 Mermaid 图表
- 嵌入 Chart.js 图表
- 游戏循环可视化
- 支柱实现程度可视化

### 4. 输出格式

- **HTML**：用于浏览器预览
- **PDF**：通过浏览器打印功能生成

### 5. 使用场景

- 向他人展示游戏分析结果
- 存档游戏拆解报告
- 生成可分享的文档

## 常见问题

### Q: 可视化不够丰富
**A**: 可以考虑集成 Mermaid/Chart.js

### Q: PDF 生成质量
**A**: 使用浏览器打印功能，设置合适的纸张大小和边距
