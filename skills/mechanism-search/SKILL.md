# Mechanism Search

从 jizhi 机制库中搜索符合条件的机制。

## 触发条件

用户消息包含以下关键词：
- 搜索机制
- 找机制
- 有没有XXX机制
- 帮我找XXX
- 机制库

## 工作流程

### 1. 解析搜索需求

从用户消息中提取搜索关键词：
- 提取核心标签：如"社交"、"三选一"、"成长"等
- 转换为英文标签搜索

### 2. 搜索机制库

机制库路径：`/root/.openclaw/agents/jizhi/workspace/mechanisms/`

**搜索方式**：
1. 遍历所有游戏目录
2. 读取每个 .md 文件
3. 匹配以下字段：
   - 核心标签
   - 机制名称
   - 一句话解释
   - 机制描述

**重要**：搜索到匹配文件后，必须读取文件内容验证是否真正符合需求，不能仅根据文件名判断。

### 3. 返回结果

按以下格式输出：

```
📋 机制搜索结果：[关键词]

找到 X 个相关机制：

### 1. [机制名称]
- 游戏：[游戏名]
- 标签：[核心标签]
- 一句话：[一句话解释]
- 路径：`mechanisms/[游戏]/[机制].md`

### 2. ...

---
如需了解详情，我可以读取具体文件内容
```

## 示例

**用户输入：**
> 帮我找找有没有社交相关的机制

**处理：**
1. 关键词：社交
2. 搜索机制库中包含"社交"标签的机制
3. 返回匹配结果

## 注意事项

- 搜索不区分大小写
- 支持中文和英文标签
- 如果没有匹配结果，告诉用户并建议其他搜索词
- 可以同时搜索多个关键词
