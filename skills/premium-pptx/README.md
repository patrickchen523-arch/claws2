# premium-pptx

> 生成排版精美、告别"AI 感"的 PowerPoint 演示文稿

---

## 这个 Skill 是什么？

`premium-pptx` 是一个融合了 **设计美学思维** 与 **PPT 技术规范** 的复合型 Skill，专门用于生成视觉品质极高的 `.pptx` 文件。

### 它解决了什么痛点？

**`pptx` skill 的局限：**
- 技术规范完整，但设计能力基础——生成的 PPT 有浓重的"AI 模板感"
- 布局单一，缺乏设计决策流程，难以生成真正好看的视觉效果
- 没有品牌风格的专项支持

**`frontend-design` skill 的局限：**
- 设计美学极强，但输出的是 HTML/CSS，**无法生成可编辑的 .pptx 文件**
- 没有技术规范约束，坐标计算、颜色格式等常见陷阱容易踩
- 没有 QA 验证流程，生成后是否正确全凭运气

**`premium-pptx` 取两者之长：**
- ✅ 像 `frontend-design` 一样先做设计决策，确保视觉品质
- ✅ 像 `pptx` 一样严格遵守技术规范，确保文件可用
- ✅ 内置完整 QA 流程 + 实战踩坑防护，生成即可用
- ✅ 新增网易品牌风格专项支持，覆盖内部汇报场景

`premium-pptx` 将这两者打通：**先像设计师一样思考，再像工程师一样精确落地。**

## 🔴 网易品牌风格（重点支持）

针对网易内部场景深度定制，完整还原网易 PPT 设计语言：

| 设计元素 | 规范 |
|---------|------|
| 主色 | 网易红 `#A80D0B`，封面/章节/结尾页全红背景 |
| 内容页 | 白色背景，标题下红色英文副标题 |
| 视觉母题 | 红色圆形数字编号，贯穿所有内容页 |
| 字体 | 中文微软雅黑 + 英文 Calibri |
| 页脚 | 每页底部红色分隔线 + `網易 NetEase` Logo + 页码 |
| 章节过渡 | 全红背景 + 大水印数字 + 白色标题 |

**内置完整页面模板：**
封面页 → 目录页 → 章节过渡页 → 标准内容页（双列要点）→ 四宫格 → 图文混排 → 时间轴 → 数据图表 → 结尾页

**适用场景：** 晋升汇报、项目复盘、季度总结、产品评审、内部培训等一切网易内部场景。

## 核心能力

### 🎨 设计先行
每次生成前先完成设计决策：确立风格方向、配色方案、字体组合、视觉母题，绝不直接上手写代码。

### 📐 布局多样
强制要求每页使用不同布局，内置 8 种高质量布局模板：左文右图 / 大数据卡片 / 2×2 网格 / 全出血背景 / 时间线流程 / 对比两列 / 图标文字行 / 引语金句

### 🛡️ 实战踩坑防护
内置所有已知技术陷阱的完整防护规则，经真实项目验证：

| 陷阱 | 说明 |
|------|------|
| 画布尺寸用错 | `LAYOUT_16x9` 实际是 `10"×5.625"`，坐标越界内容全部挤在角落 |
| 颜色带 `#` 号 | `"#FF0000"` 会静默损坏文件，必须用 `"FF0000"` |
| 圆圈+文字错位 | 必须同 y 同 h，加 `valign: "middle"`，否则文字换行偏移 |
| 时间轴节点偏移 | 用 `(i+0.5)*nodeW` 严格等分，不可手动偏移 |
| 页脚被截断 | 用 `H-0.45` 相对计算，不硬编码绝对坐标 |
| 中文弯引号 | `""` 会导致 JS 语法错误 |

### ✅ 强制 QA
- **生成前**：6 条自检清单逐一核对
- **生成后**：转图片逐帧排查，问题归零才算完成
