# 战棋游戏拆解分析

## 品类概述

战棋游戏（Turn-Based Strategy，简称TBS）是策略游戏的重要分支，以回合制战斗为核心，结合角色养成与剧情叙事。战棋游戏在2020年代经历了复兴，尤其是手游领域，紫龙游戏凭借《梦幻模拟战》《天地劫》等产品证明了这一品类的商业价值。

---

## 一、端游（5款）

### 1. 火焰纹章：风花雪月 (Fire Emblem: Three Houses)

**基础信息**
- 开发商：Intelligent Systems / 任天堂
- 上线时间：2019年7月26日（Switch）
- 销量：全球累计412万份（截至2022年12月）
- 特点：战棋玩法 + 校园生活模拟 + 多线剧情

**X-Statement公式**
【SRPG + 校园生活 + 多线叙事】→ 玩家在魔法学院中培养角色，参与战斗，体验三国纷争的史诗剧情

**核心支柱**
- 支柱1：【角色培养】→ 【职业养成 + 技能继承】→ 每角色独立转职树，支持多职业发展
- 支柱2：【战术策略】→ 【地形利用 + 兵种相克】→ 高低差地形影响攻击闪避，不同职业有属性克制
- 支柱3：【剧情选择】→ 【三条路线 + 伙伴支援】→ 三个国家势力选择，支援对话积累好感

**核心体验**
- 以秒为单位：3D等距视角，玩家在战场上移动角色，使用武器/魔法攻击，享受策略博弈乐趣
- 以小时为单位：主线关卡（20-30小时）→ 支援对话培养 → 自由活动经营 → 多周目体验不同剧情

## 三、战棋品类设计洞察

### 核心共性

1. **战斗系统是核心**
   - 兵种相克（步兵→枪兵→骑兵→步兵）
   - 地形影响（山地增防、森林隐蔽、水域限制）
   - 职业系统多样性

2. **养成深度决定长线**
   - 转职树设计（职业选择带来差异化）
   - 技能继承（养成积累）
   - 装备驱动

3. **剧情是加分项**
   - 战棋玩家重视剧情体验
   - 多结局/多周目设计提升重玩价值

### 商业化模式

- **抽卡/角色获取**：主要付费点
- **战令/通行证**：长线活跃付费
- **体力限制**：控制资源产出

### 设计趋势

1. **轻量化**（火焰纹章英雄）vs **深度化**（皇家骑士团2）分化
2. **画面升级**（HD-2D、3D化）
3. **国风融合**（天地劫、钢岚）
4. **IP改编**（经典IP手游化）

## 四、可入库机制筛选

基于上述分析，以下机制值得入库：

### 1. 兵种相克系统
- 设计：基础循环（步兵克枪兵，枪兵克骑兵，骑兵克步兵）
- 变体：属性相克（冰克雷、雷克火、火克冰）
- 核心价值：策略深度的基础

### 2. 地形影响机制
- 设计：地形提供增伤/减伤/特殊效果
- 核心价值：增加战场策略维度

### 3. 转职树系统
- 设计：每角色独立转职树，职业选择带来差异化
- 核心价值：角色养成深度

### 4. WT/行动顺序系统
- 设计：基于权重的时间轴行动顺序
- 核心价值：增加策略变数

### 5. 多线剧情/多结局
- 设计：玩家选择影响剧情走向和结局
- 核心价值：提升重玩价值

### 6. 指挥官/CO技能系统
- 设计：特定角色拥有独特技能，可改变战局
- 核心价值：增加策略爆发点
